Chipledger Documentation

About this Project

This app is designed to tracking money when playing poker/blackjack games.

User Features

  1. Add individual players.
  2. Track buyins by payment type. (cash, zelle, cashapp, venmo, etc)
  3. Cashout by payment type.
  4. History in timeline, showing in order when players join the table, buyin, and cashout.
  5. See total pot, total buy in, and cashout amounts.

Developer Features

  1. Uses only open source software.
    1. Apache for file permissions and routing.
    2. PHP for API and routing.
    3. JS for frontend rendering and routing.
    4. SQLite for database.
    5. Bootstrap and Bootswatch 5 for components.
  2. Can be run locally or on another hosting provider.
  3. Fully open source.
  4. No AI written code. All chipledger software is written by hand.
