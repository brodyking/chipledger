# Chipledger

Chipledger is a web app that tracks poker buy ins and cashouts for home poker games.

## Requirements
This project uses and requires:

- **php 8.3.14**
- **apache**
- **sqlite**
- **bootstrap**

## Install

```bash
git clone https://github.com/brodyking/chipledger.git
```
